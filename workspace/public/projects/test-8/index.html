Docker + Nginx: App Estática

Containerización de una aplicación HTML estática con servidor Nginx

Descripción de la Consigna

Generar un Dockerfile para servir una aplicación HTML estática usando Nginx como servidor web. Incluir archivo docker-compose.yml para facilitar el despliegue. La configuración debe:

  • Copiar archivos de la app al directorio de Nginx
  • Usar imagen base ligera (Alpine)
  • Configurar health checks
  • Soportar compresión gzip
  • Crear la app si no existe

Pasos Ejecutados

  1. Creé la estructura de directorios: workspace/apps/todo-app/docker/
  2. Creé una Todo App HTML mínima con diseño moderno, almacenamiento local y diseño responsive
  3. Escribí el Dockerfile usando nginx:alpine como base
  4. Configuré nginx.conf con compresión gzip, cache de estáticos y endpoint de health
  5. Creé docker-compose.yml con healthcheck y mapeo de puertos
  6. Generé esta página de documentación

Archivos Generados

/workspace/apps/todo-app/index.html

Aplicación Todo List con diseño moderno, almacenamiento localStorage,
transiciones CSS y diseño responsive. Incluye funcionalidad CRUD completa.

/workspace/apps/todo-app/docker/Dockerfile

FROM nginx:alpine RUN rm -rf /usr/share/nginx/html/* /etc/nginx/conf.d/* COPY . /usr/share/nginx/html COPY docker/nginx.conf /etc/nginx/conf.d/default.conf EXPOSE 80 CMD ["nginx", "-g", "daemon off;"]

/workspace/apps/todo-app/docker/nginx.conf

server { listen 80; server_name localhost; root /usr/share/nginx/html; index index.html; gzip on; location / { try_files $uri $uri/ /index.html; } location /health { return 200 "healthy\n"; } }

/workspace/apps/todo-app/docker/docker-compose.yml

version: '3.8' services: todo-app: build: context: .. dockerfile: docker/Dockerfile ports: - "8080:80" healthcheck: test: ["CMD", "wget", "--spider", "http://localhost/health"] interval: 30s

Comandos de Uso

# Construir y ejecutar con docker-compose cd /workspace/apps/todo-app/docker docker-compose up --build # O con Docker directamente docker build -f docker/Dockerfile -t todo-app . docker run -p 8080:80 todo-app # Verificar health curl http://localhost:8080/health
